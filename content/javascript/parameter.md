## 💙 매개변수와 인수

### 1. 매개변수 (Parameters)

```js
// a와 b는 매개변수
function sum(a, b) {
  return a + b;
}
```

함수 안에서 사용하는 데이터를 받아주는 매개체가 되는 변수<br />
함수 안에서만 사용이 가능

### 2. 인수 (Arguments)

```js
let a = sum(1, 2); // 1과 2는 인수
```

함수의 매개변수가 받아 줄 특정한 데이터
