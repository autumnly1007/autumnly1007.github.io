## 💙 변수와 예약어

### 1. 변수 (Variable)

데이터를 지정하고 참조(사용)하는 데이터의 이름

- var: 사용하지 않는 것이 권장됨 (호이스팅 문제)
- let: 재할당 가능
- const: 재할당 불가능

### 2. 예약어 (Reserved Word)

특별한 의미를 가지고 있어, 변수나 함수 이름 등으로 사용할 수 없는 단어

if, else, false, true, while, this, break ...
